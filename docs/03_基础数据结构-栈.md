# 栈 (Stack)

栈是一种**后进先出(LIFO)**的数据结构，常用于递归和回溯问题的解决。栈允许操作**栈顶元素**，**栈顶元素**是栈所着重关注的点。

## 特性

- **LIFO (后进先出)**：最新加入的数据最先被移除。
- **栈顶操作**：只能对栈顶元素进行操作。

## 优缺点

### 优点

- **操作简单**：只需处理栈顶元素，操作直观。
- **适合递归和回溯**：如因数调用、表达式求值。

### 缺点

- **只能访问栈顶**：无法访问中间或底部的元素。
- **空间浪费**：如果栈的大小没定过大，则会浪费内存。

## 代码示例

```javascript
const stack = [];
stack.push(1);
stack.push(2);
stack.push(3);
console.log(stack);   // [1, 2, 3]
const top = stack.pop();
console.log(top);     // 3  ← 后进先出
```
