# 队列 (Queue)

队列是一种**先进先出(FIFO)**的数据结构，常用于广度优先搜索、任务调度等场景。队列允许在**队尾入队**、在**队首出队**，**队首元素**是队列所着重关注的点。

## 特性

- **FIFO (先进先出)**：最先加入的数据最先被移除。
- **队首与队尾**：只能从队首取元素，从队尾加元素。

## 优缺点

### 优点

- **公平有序**：按加入顺序依次处理，符合“先来先服务”。
- **适合 BFS 与调度**：如层序遍历、任务队列、消息队列。

### 缺点

- **只能访问队首**：无法随机访问中间或队尾元素（除非遍历）。
- **固定顺序**：不能像优先队列那样按优先级出队（需用堆等结构）。

## 代码示例

```javascript
const queue = [];
queue.push(1);
queue.push(2);
queue.push(3);
console.log(queue);   // [1, 2, 3]
const first = queue.shift();
console.log(first);   // 1  ← 先进先出
```
